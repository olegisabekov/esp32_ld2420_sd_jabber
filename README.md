# esp32_ld2420_sd_jabber
  Perimeter security control sensor (ld2420) with messaging via xmpp protocol, settings stored on SD card.

# Dependencies

  - IniFile https://github.com/stevemarple/IniFile
  - XMPP_StartTLS https://github.com/olegisabekov/XMPP_StartTLS
  - LD2420 https://github.com/cyrixninja/LD2420

# Hardware Requirements

  - Arduino board (Uno, Nano, ESP32, etc.)
  - HLK LD2420 radar sensor
  - CDCard reader
  - Connecting wires
